# SIOMS - Full Stack Project

## Quick Start

**Terminal 1 - Backend:**
```bash
cd sioms-backend
npm install
npm run dev
# Runs on http://localhost:5000
```

**Terminal 2 - Frontend:**
```bash
cd sioms-frontend
npm install
npm run dev
# Runs on http://localhost:3000
```

Open http://localhost:3000 and login with any email + password `admin123`

See `sioms-backend/README.md` for full API documentation.
